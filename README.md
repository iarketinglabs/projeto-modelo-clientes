# Projeto Modelo · Atômica

> 🏗️ **Starter kit** para novos projetos de cliente.
> Clone, configure, substitua os placeholders e comece a construir.

---

## O que é isto?

Este repositório é o **molde canônico** da Atômica para iniciar projetos de cliente. Ele contém:

- 📂 **Estrutura de diretórios** padronizada
- 🧠 **Framework DOE** (Diretivas → Orquestração → Execução)
- 🏭 **Processo Atômica** de 6 fases + 7 gates
- 🤖 **Agentes de IA** pré-configurados (`.agents/`)
- 🔧 **Stack tecnológica default** (Python, Next.js, PostgreSQL, Docker)
- 🔒 **Regras de segurança e qualidade** inegociáveis
- 📋 **Placeholders** em `directives/` prontos para serem preenchidos com o conteúdo do cliente

---

## 🚀 Comece aqui

Se você é um **membro da equipe Atômica** iniciando um novo projeto:

👉 **Leia a pasta [`orientacao/`](orientacao/)** — contém tudo que você precisa:

| Arquivo | Conteúdo |
|---------|----------|
| [`orientacao/sobre-o-template.md`](orientacao/sobre-o-template.md) | Framework DOE, stack default, regras de ouro |
| [`orientacao/guia-onboarding.md`](orientacao/guia-onboarding.md) | Passo a passo prático para iniciar projetos |

## Estrutura

```
projeto-modelo-clientes/
├── orientacao/              # ★ Documentação do template (leia primeiro!)
├── directives/              # ⚠️ Placeholders — substituir pelo conteúdo do cliente
│   ├── PRD.md               #     PRD do projeto do cliente
│   ├── architecture.md      #     Arquitetura do cliente
│   └── brand_voice.md       #     Voz da marca do cliente
├── .agents/                 # Agentes, skills, comandos (herdados)
├── executions/              # Onde o código do cliente vai morar
└── tmp/                     # Arquivos temporários
```
